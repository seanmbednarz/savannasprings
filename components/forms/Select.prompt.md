Dropdown select styled to match Input. Use for choosing a product, service type, or location.

```jsx
<Select label="I'm interested in" placeholder="Choose a product">
  <option>Water softener</option>
  <option>Drinking water (R.O.)</option>
  <option>Salt delivery</option>
</Select>
```

Sizes `sm`/`md`/`lg`; supports `hint` and `error`.
